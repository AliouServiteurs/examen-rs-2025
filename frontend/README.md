# Frontend - React Material-UI

## Structure
```
src/
├── components/
│   ├── Common/         # Composants réutilisables
│   ├── Layout/         # Header, Layout
│   └── Personne/       # Gestion personnes
├── services/           # API calls
├── theme/              # MUI Theme
└── App.js
```

## Installation
```bash
npm install
```

## Démarrage
```bash
npm start
```

Application disponible sur http://localhost:3000

## Technologies

- React 18
- Material-UI (MUI)
- Apollo Client (GraphQL)
- Axios (REST)
